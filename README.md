# app-ssl-lab

本仓库用于归档和发布多个独立 HTML 页面。这些页面通常与 SSL/TLS 相关的演示、测试或学习资料有关，但并不针对任何特定域名或具体网站。

## 项目简介

`app-ssl-lab` 是一个轻量级的页面集合仓库。你可以把它看作一个静态页面归档中心，主要存放与 SSL/TLS 协议、证书验证、加密通信等主题相关的 HTML 文件。每个 HTML 页面都是独立的，可以直接在浏览器中打开查看，或通过 GitHub Pages 等服务进行预览。

## 目录说明

仓库的根目录下直接存放 HTML 文件，没有复杂的嵌套结构。建议按以下方式组织文件：

- 每个 HTML 文件使用有意义的名称，例如 `ssl-check-demo.html` 或 `cert-viewer.html`。
- 如果文件数量增多，可以考虑按功能或主题创建子目录（如 `demo/`、`tool/`、`doc/`），但当前版本保持扁平化。

## 页面归档说明

本仓库中的 HTML 页面均以独立形式归档，彼此之间没有依赖关系。每个页面应包含完整的 HTML 结构（包括 CSS 和 JavaScript，如果有），以确保离线或在线环境下都能正常工作。

- 页面内容仅限于技术演示、示例或说明，不涉及任何商业推广或具体网站的引用。
- 所有页面均基于公开的技术规范或通用实践编写，不包含敏感信息或私人数据。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面，但请确保内容与 SSL/TLS 相关，且不包含任何具体域名、网址、关键词或营销承诺。
- 提交前请检查页面是否能在常见浏览器中正常渲染，并避免引入外部资源（除非是公共 CDN 上的稳定库）。
- 仓库维护者会定期检查页面内容，清理失效或过时的文件。

## 如何使用

1. 直接点击仓库中的 HTML 文件，浏览器会显示其内容（如果仓库启用了 GitHub Pages，也可以通过 `https://<用户名>.github.io/app-ssl-lab/<文件名>` 访问）。
2. 你也可以将仓库克隆到本地，在本地服务器环境中打开这些 HTML 文件。
